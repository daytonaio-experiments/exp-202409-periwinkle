Instructions for Writing a Prompt

1. Create a prompt that guides an AI to generate BDD scenarios based on a given user story.

2. Format: Use the provided format for your prompt:
   - As a [type of user], I want [an action] so that [a benefit/a value]

3. Ensure your prompt includes:
   - Relevant details (e.g., user roles, actions, expected results).
   - Example: As a user, I want to log in to my account so that I can access personalized content.
